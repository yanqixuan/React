# React 入门

1. cnpm i create-react-app -g  安装脚手架
2. create-react-app helloreact 建立模板文件

## 组件
App.js
Child.js

### 子组件
子组件的方法 如果是普通函数 this会丢失，绑定不到。 使用bind（this) 绑定
使用箭头函数不会丢失

## render
组件需要渲染的页面

## jsx
jsx 是一个js的语法拓展
react 自己的语法  js+html

## state
组件内部的数据

## props
外部传入的数据
在父组件引入子组件  传入msg 子组件直接使用

## {}
表示其中内容需要运算
